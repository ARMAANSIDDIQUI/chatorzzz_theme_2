# Chatorzzz - Premium Sweets from Moradabad

The ultimate e-commerce destination for premium candies and sweets, based in Moradabad.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
